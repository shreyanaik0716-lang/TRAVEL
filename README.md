## ✨ Features
- 🎥 **Background Video Banner** on homepage
- 🏨 **Packages Section** with hotel details and pricing
- 📍 **Locations Section** highlighting Mysore, Coorg, Hampi, Chikmagalur, Gokarna, Udupi, Shivamogga, and Badami
- 🖼️ **Gallery Section** with traveler photos
- 📧 **Newsletter Subscription**
- 📝 **Contact Form** with name, email, subject, and message fields
- 🔗 **Quick Links & Social Media Integration** in footer

## 🚀 How to Run
1. Clone or download the repository.
2. Ensure the folder structure remains intact (`css/`, `files/`).
3. Open `index.html` in any modern browser.
4. Navigate through the pages using the navbar.
